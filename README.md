# MVPDemo
通过使用 retrofit + okhttp + rxjava + rxandroid 底层逻辑功能封装实现mvp网络请求

封装支持如下功能：

1. 常规的post，get请求等
2. 文件上传和进度监听
3. 文件下载和进度监听
4. 自定义异常处理模块

**说明：** 由于原本使用的是公司接口测试，发布后仅提供部分可用测试案例
